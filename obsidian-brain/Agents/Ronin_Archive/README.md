# Ronin Archive — Cold Storage

Dormant, unassigned agents serialized as state-vectors with their accumulated
skill-trees preserved (spec §2.2, "Cold Storage (The Ronin)").

Each dormant agent is one Markdown file in this directory: front-matter carrying
the serialized state-vector, followed by a human-readable skill-tree. Reawakened
by the swarm when a matching task appears.
